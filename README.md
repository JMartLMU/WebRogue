## Professional Elevator Pitch
WebRogue is a domain-specific language for browser-based roguelike game development. The language is designed around declarative world construction, entity definitions, and rule-driven turn behavior, with a compiler that performs static analysis before generating browser execution. Rather than treating game logic as raw JavaScript, WebRogue introduces language-level constructs for maps, entities, scene flow, and rendering, allowing compile-time checks for semantic errors such as undefined references, invalid map structure, unreachable states, and inconsistent game-state usage.

## Story
Game Development is a really big passion of mine. I remember how frustrating it was trying to use Pygame as an introduction to programming games. It was clunky at rendering, setting up maps, and hard to share. I want to create a language to similify the process into an easily excessible, lightweight, web-based structure. To provide an easy way to make games on the web.
